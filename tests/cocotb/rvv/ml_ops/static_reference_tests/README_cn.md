# 参考与冻结的 ML 运算测试（Reference and Frozen ML Operations Tests）

本目录包含一套完全冻结、自包含的测试 target，用作**功耗分析（power analysis）**和**门级网表仿真（gate-level netlist simulation）**的稳定基准。

## 动机
标准回归测试被设计为运行时可配置，以覆盖各种矩阵维度。然而，这种灵活性给性能和功耗分析带来了风险：
- 对测试形状（shape）或参数的更改会改变仿真周期数和功耗曲线，从而悄无声息地使基线测量结果失效。
- 对通用 runner 逻辑或编译器优化标志的修改会影响指令调度和执行指标。

为防止这种情况，本文件夹内的测试被锁定：
- Runner 二进制将形状维度硬编码为常量，并静态分配内存。
- 输入使用确定性的伪随机种子生成。
- 文件名显式包含测试维度（例如 `float_matmul_16x48x16`）。

> [!WARNING]
> 请勿修改本目录中现有测试用例的代码、矩阵维度或编译选项。如果需要新的形状或配置，请在现有文件旁边创建新的源文件，将其重命名以反映维度，并在 BUILD target 中定义一个新的测试用例。
