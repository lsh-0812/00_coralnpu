# 如何贡献

我们非常欢迎你为本项目提交补丁和贡献。你只需遵循以下几条简单的准则。

## 贡献者许可协议（Contributor License Agreement）

向本项目的贡献必须附带一份贡献者许可协议（CLA）。你（或你的雇主）保留对你贡献内容的版权；该协议只是授予我们使用和重新分发你贡献内容的许可，作为本项目的一部分。请前往 <https://cla.developers.google.com/> 查看你当前已签署的协议，或签署一份新协议。

通常你只需提交一次 CLA，因此如果你已经提交过（即使是为其他项目提交的），你很可能无需再次提交。

## 代码审查（Code Reviews）

所有提交（包括项目成员的提交）都需要经过审查。我们为此使用 Gerrit 代码审查工具。更多信息请参阅
[Gerrit 用户指南](https://gerrit-documentation.storage.googleapis.com/Documentation/3.8.2/intro-user.html)。

## 社区准则

本项目遵循
[Google 开源社区准则](https://opensource.google/conduct/)。
